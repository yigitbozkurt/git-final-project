# Simple Interest Calculator

This project is a simple shell script designed to calculate simple interest based on principal amount, annual interest rate, and time period in years.

## Project Structure
- `simple-interest.sh`: The main script that performs the calculation.
- `LICENSE`: Apache 2.0 license file.
- `CODE_OF_CONDUCT.md`: Guidelines for project participants.
- `CONTRIBUTING.md`: Instructions on how to contribute to the project.

## How to use
1. Clone the repository.
2. Run the script using `bash simple-interest.sh`.
3. Enter the required values when prompted.
